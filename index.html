Galaxy Zoo Model

A simple project page for my galaxy classification project built with PyTorch. The model uses galaxy images to predict the main galaxy shape and whether the galaxy is edge-on.

Project Summary

About

This project explores galaxy morphology classification using images from the Galaxy Zoo dataset. It focuses on two of the five Galaxy Zoo questions and uses a ResNet18-based PyTorch model.
